\section{Hydrologic Modeling}
\subsection{(a)}
For each time $t$ and each geographic point $(x,y)$, the state variable $\mathbf{y}_t$ is defined as
\[
\begin{pmatrix}
  \text{SWE} \\
  T_{\text{snow}}
\end{pmatrix}_{x,y}
\]
where $\text{SWE}$ is snow water equivalent and $T_{\text{snow}}$ is snowpack temperature.
